Mixminion is the standard implementation of the Type III anonymous remailer protocol.

The Design Document gives our justifications and security analysis for the Mixminion design:

  • PostScript version
  • PDF version
  • Latex source
  • Bibtex file
  • Roger's design overview slides (PDF)

The Spec aims to give developers enough information to build a compatible version of Mixminion:

  • Latex/text version

The mixminion-dev mailing list is where a lot of the discussion happens. Please look at its archives and consider subscribing. There's also a list for cvs commits.

And you can browse the Minion cvs:

  • ViewCVS
  • Regularly updated cvs sandbox
  • anonymous pserver access (password is guest):
    • cvs -d :pserver:guest@cvs.seul.org:/home/minion/cvsroot login
    • cvs -d :pserver:guest@cvs.seul.org:/home/minion/cvsroot co src doc
